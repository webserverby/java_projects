    - ФОРМАТ PROPERTIES -
 В Java, это стандартный файл хранения настроичной информации.

    - УРОВНИ ЛОГИРОВАНИЯ -
 logger.trace() - когда в программе не произошло ничего весомого (Обыкновенный отчет)
 logger.debug() -
 logger.info() - когда в программе произошло какое-то решение (Например условие и от него зависит дальнейший результат)
 logger.warn() - когда произошло исключение, но программа была расчитана для него (То есть ошибка не критична)
 logger.error() - когда произошла не предвиденная проблема
 logger.fatal() -

    - СИНТАКСИС -
 В log4j, каждая новая строка это пара: ключ = значение
 # Комментарий
 variable = "value1";
 log4j.(...) = (...) # Всё что начинается с log4j это уже обращение к инструкциям самого log4j
 log4j.(...) = ${variable}/(...) # Чтобы обратится к объявленной переменной: ${name_variable}
 # Level - нужно внести один из уровней логирования, после идет перечисление аппендеров
 log4j.rootLogger = Level, VARIABLE1, VARIABLE2
